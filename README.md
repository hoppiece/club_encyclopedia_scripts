# club_encyclopedia_scripts

# 概要
2019年度新入生サポート事務局のサークル大百科制作にて，掲載団体を登録するGoogle Formから各団体に自動で案内を返信するためのGoogle Apps Scripts.

# 使い方
1. フォームの解答をスプレッドシートに出力

2. スプレッドシートのツールからスクリプトエディタを開き，本コードをコピペ

3. 関数のトリガーを解答を受け取ったときに設定
